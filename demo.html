🛡️ AWS Security Group Manager

Amazon Q가 1시간 만에 개발한 엔터프라이즈급 웹 애플리케이션

📊

실시간 대시보드

Security Group 현황, 만료 예정 규칙, 요청 통계를 실시간으로 모니터링하세요. 그라데이션 카드와 애니메이션으로 시각적 효과를 극대화했습니다.

🔐

Security Group 관리

AWS Security Group을 실시간으로 동기화하고 관리하세요. 만료일 자동 설정, 규칙 추가/삭제, 상태 모니터링을 지원합니다.

👥

승인 워크플로우

사용자 요청 → 관리자 승인 → 자동 적용 워크플로우로 안전하고 체계적인 Security Group 관리를 실현합니다.
